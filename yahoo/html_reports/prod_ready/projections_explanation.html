üéØ What the Scores Represent

Fantasy Value Score Explanation

Each player receives a Fantasy Value Score representing their total projected contribution across all 9 Yahoo categories:

  • Points, Rebounds, Assists - Raw production normalized against league averages
  • Steals, Blocks, 3-Pointers - Defensive and shooting contributions
  • Field Goal %, Free Throw % - Efficiency metrics (higher is better)
  • Turnovers - Ball security (lower turnovers = higher score)

Score Ranges:

700+: Elite tier (Jokic, Luka level) - League winners
600-699: All-Star tier - First round picks
500-599: Solid starter - Early-mid round targets
400-499: Role player - Mid-late round value
300-399: Streaming option - Deep league relevant
200-299: Waiver wire - Emergency fill-in
